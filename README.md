 ## 三、研究方向与技术路线
 
 我的研究围绕一条清晰的主线展开：以王聪教授提出的确定学习（Deterministic Learning）理论为核心方法，通过对心电信号的动力学建模与分析，解决心血管疾病早期精准诊断中的关键难题。在此基础上，研究进一步从心脏病学跨域拓展到肿瘤治疗相关心脏损伤（RIHD）检测这一前沿交叉方向，体现了心电动力学图（CDG）方法在不同病因心脏损伤检测中的通用价值。研究大致经历了以下阶段。
 
 ### 3.1 第一阶段：心电动力学图（CDG）的临床转化与验证（2020-2022年）
 
 心电动力学图（Cardiodynamicsgram, CDG）是王聪教授团队提出的一种心电信号分析新方法（Wang et al., Science China Information Sciences, 2016），其核心思想是利用确定学习算法对心电信号ST-T段进行非线性动力学建模，提取隐藏在常规心电图中的微弱动力学信息并三维可视化。常规心电图对心肌缺血的诊断敏感性不足（约60%的缺血患者心电图表现正常或大致正常），CDG的目标就是解决这一长期临床难题。
 
 我的第一项主要工作是将CDG方法推向大规模临床验证（自动化学报, 2020）。为此，我在中国医学科学院阜外医院和新疆石河子市人民医院两个中心收集了781例经冠脉造影检验的疑似心肌缺血患者数据，建立了当时国内规模较大、且同时包含心电图正常/大致正常患者、冠脉阻塞性与非阻塞性病变患者的心肌缺血数据集。利用该数据集，我验证了CDG在心电图正常或大致正常患者中检测心肌缺血的能力，达到灵敏度90.1%、特异度85.2%、AUC 0.93。特别有意义的是，CDG具有明确的物理意义（代表心室复极离散度变化率），使得我得以对假阳性病例进行逐例分析，发现许多假阳性患者存在冠脉慢血流现象——这是一种被临床漏诊的真实心肌缺血，揭示了数据标注的系统性偏差。
 
 延续这一方向，我进一步将CDG与集成学习相结合（Computer Methods and Programs in Biomedicine, 2022, IF=6.4, 中科院2区），把验证范围扩展到三个独立医学中心（阜外医院、齐鲁医院、石河子医院），共499例非诊断性心电图患者。利用CDG提取的频谱拟合指数、Lyapunov指数和Lempel-Ziv复杂度等动态特征，通过Bagging异构集成（SVM-RBF、SVM-Linear、Boosting Tree加权投票）构建心肌缺血检测模型，准确率89.10%，并在PTB公开数据集上验证了模型的泛化能力（91.11%）。同期，我还在Physiological Measurement上发表了基于多尺度分解和混合特征选择的CDG动态特征提取方法，在PTB数据集上实现94.75%的MI检测准确率，并在齐鲁医院200例独立临床数据上验证了84.96%的准确率（Physiol. Meas., 2022）。
 
 此外，我与齐鲁医院急诊科合作，利用CDG评估急性冠脉综合征（ACS）患者经皮冠脉介入术（PCI）的疗效（中华急诊医学杂志, 2022）。在203例ACS患者中，CDG在PCI术后由散乱变为规整，90.28%术后无严重残余狭窄患者的CDG指标显著降低（P<0.001），表明CDG可直观反映心肌灌注的改善情况。
 
 ### 3.2 第二阶段：跨模态验证与临床应用探索（2023年）
 
 2023年的一项重要工作是利用冠脉CT血管成像（CCTA）和CT血流储备分数（CT-FFR）验证CDG的诊断价值（Clinical Cardiology, 2023）。我与山东省立医院影像科合作，纳入456例疑似冠心病患者，同时进行CCTA检查与常规心电图检查。以CCTA>=50%狭窄或CT-FFR<=0.8为冠脉疾病金标准，CDG的诊断准确率79.56%，AUC=0.836；在223例心电图诊断阴性亚组中，准确率80.27%，AUC=0.842。更重要的是，CDG值与CT-FFR显著相关（r=-0.395, P<0.001），表明CDG不仅反映冠脉狭窄的解剖信息，还能部分反映心肌缺血的功能信息。
 
 ### 3.3 第三阶段：动静特征融合与可解释性深化（2024年——学术高产年）
 
 2024年是我学术产出最密集的一年，发表6篇论文，核心思路是实现心电图"静态度量"与"动态特征"的有机融合，并提升模型的可解释性。
 
 - **动静特征融合的心律失常分类**（IEEE Trans. Instrum. Meas., 2024, IF=7.0, JCR Q1）：将CDG动态特征与QT间期等静态特征通过DS证据理论融合，4类心律分类准确率91.71%，7类88.93%，融合后分别提升4.59%和5.26%。
 - **多尺度联合递归量化分析（MSJRQA）**（IEEE Trans. Instrum. Meas., 2024, IF=7.0）：将确定学习获取的动态信息与ECG信号通过VMD多尺度分解，引入联合递归量化分析融合时空与动态特征，MI检测准确率94.50%，区分MI与非MI但伴ST-T异常患者准确率90.78%。
 - **多相动态特征与病因鉴别**（Biomedical Signal Processing and Control, 2024, IF=5.7, 中科院2区）：首次实现基于心电图的心肌缺血病因鉴别——区分冠脉狭窄（COAS）和冠脉慢血流（CSF）。缺血检测AUROC=0.9126，病因鉴别AUROC=0.9238。
 - **StackTree可解释集成树**（Physiol. Meas., 2024）：提出栈式集成方案简化随机森林，在保持97.1%准确率（PTB）的同时生成可理解的决策规则和可视化分类逻辑。
 - **Shapelets可解释MI检测**（Physiol. Meas., 2024）：将深度学习的shapelets引入CDG动力学信号进行局部模式提取，准确率94.11%。
 
 ### 3.4 CDG跨域拓展：肿瘤放射治疗心脏损伤的早期检测（2024年）
 
 2024年发表的一项特殊且意义重大的工作是利用CDG检测放射性心脏损伤（Radiation-Induced Heart Damage, RIHD）（Journal of Cardiovascular Translational Research, 2024）。这项工作将CDG从传统的冠心病领域跨域拓展到了肿瘤治疗相关心血管毒性检测——一个快速发展的前沿交叉学科（Cardio-Oncology）。
 
 放疗是胸部恶性肿瘤（如肺癌、食管癌、乳腺癌）的重要治疗手段。放疗技术的进步使肿瘤患者生存期显著延长，但越来越多的临床证据表明，放疗后心脏损伤的长期风险显著增加。然而目前临床上缺乏简单、敏感的手段来早期检测RIHD——常规心电图敏感性不足，心脏活检则有创且并发症风险高。可以说，临床上迫切需要一种可靠的、无创的早期检测方法。
 
 我与山东省肿瘤医院放疗科合作，建立了Wistar大鼠胸部照射模型（单次40Gy心脏顶点照射），分别在照射后3天至6个月的多个时间点采集心电图和心脏组织。利用确定学习生成大鼠心电图的CDG，通过小波变换和熵分析量化CDG变化，以组织病理学（巨噬细胞浸润和心肌纤维化）为金标准。结果发现：
 
 1. CDG形态变化显著：放疗后CDG由规整变为不规则/混沌形态，CDG熵指标显著降低。
 2. CDG早于病理学改变：CDG变化与组织病理学显著相关，且发生在组织病理学变化之前——这意味着CDG可以在心肌出现不可逆损伤的早期阶段就发出预警，这一发现具有重要的临床意义。
 3. 方法有效性验证：CDG在RIHD检测中的有效性通过组织病理学金标准得到了严格验证，为临床转化提供了可靠的实验依据。
 
 这项工作的意义不仅在于RIHD本身，更在于它证明了CDG反映心脏电活动异常的能力不局限于缺血性心脏病，辐射、炎症等其他病因导致的早期心肌损伤同样可以在CDG上被灵敏地捕捉。这为CDG作为一种通用型心脏损伤早期检测工具奠定了基础，也为我后续将CDG拓展到冠脉慢血流筛查（2026年IEEE TBME）等更多临床场景提供了方法论信心和实验基础。在肿瘤心脏病学（Cardio-Oncology）成为心血管学科重要新兴分支的背景下，这项工作的临床转化价值尤为突出。
 
 ### 3.5 第四阶段：前沿课题突破（2025-2026年）
 
 进入2025-2026年，研究方向进一步向更具挑战性的临床前沿问题拓展。
 
 **知识驱动多导联网络（FEC-KML）**（Expert Systems with Applications, 2025, IF=9.4, JCR Q1, 中科院计算机科学1区）：针对心肌梗死检测和定位任务，我提出了一种融合临床知识的深度学习模型。利用临床经验将12导联划分为解剖导联组（对应左前降支、回旋支、右冠状动脉供血区域），设计多分支ResNet结构，同时以ECG波形和CDG动力学为输入进行联合学习。在PTB-XL大数据库上，MI检测准确率超过94%，定位准确率超过87%，优于当时最先进方法。
 
 **冠脉微血管功能障碍的无创筛查（AG-STDF）**（IEEE Trans. Biomedical Engineering, 2026, IF=4.4, 中科院2区，生物医学工程领域旗舰期刊）：冠脉慢血流（CSF）是冠脉微血管功能障碍（CMD）的典型表型，在冠脉造影中占1-7%但静息心电图无显著改变，临床诊断高度依赖有创功能学检查，是国际心血管领域的前沿难题。我提出的解剖学引导心电时空动力学融合框架（AG-STDF），将心脏冠脉灌注分区映射到心电图导联，再把动态信息（通过确定学习建模得到）和静态形态信息通过多尺度联合递归量化分析融合，实现CSF检测AUROC=0.9467，并可定量预测冠脉血流速度（CTFC预测R^2=0.5951）。
 
 **CDG-MACE评分模型**（Physiol. Meas., 2026年5月）：最新工作将CDG技术转化为临床可用的风险分层评分系统——CDG-MACE评分，面向急诊胸痛但心电图正常的患者群体，推动CDG从辅助诊断工具向临床风险量化评分工具转化。

 ## 四、核心方法：心电动力学图（CDG）
 
 我的所有工作都围绕CDG展开。CDG由王聪教授等提出（Wang et al., Science China Information Sciences, 2016），其技术要点概括如下：
 
 1. 理论基础：基于确定学习理论，通过对由非线性动力系统产生的周期性/回归性时间序列进行神经网络建模，实现系统未知动力学的局部精确逼近。
 2. 技术路线：12导联ECG → Kors变换为3导联VCG → ST-T段定位 → RBF神经网络动力学建模 → 三维可视化CDG。
 3. 物理意义：CDG代表心室复极离散度的变化率。正常人心室内电活动传导有序，CDG呈现规整螺旋形态；心肌缺血/损伤时心肌复极离散度增大、传导异常，CDG呈现散乱混沌形态。这一明确的物理含义是CDG区别于纯深度学习方法的根本优势，也为临床医生的理解与信任奠定了基础。

 ## 五、研究问题的难度与价值
 
 ### 5.1 研究难度
 
 我聚焦的问题横跨心血管疾病诊断和肿瘤治疗相关心脏损伤检测两个领域，均属于公认的困难场景：
 
 | 问题 | 难点 |
 |------|------|
 | 心电图正常/非诊断性ECG的心肌缺血检测 | 信号极其微弱（60%以上无典型ST-T改变），标注依赖有创冠脉造影，成本高风险大 |
 | 心肌缺血病因鉴别（冠脉狭窄 vs 慢血流） | 两种病因均可导致心肌缺血，但治疗方案完全不同（支架 vs 药物），心电图上缺乏明确的区分性特征 |
 | 放射性心脏损伤的早期检测 | 常规心电图敏感性不足，组织病理学有创且变化出现较晚，目前临床缺乏可靠的无创早期检测手段 |
 | 心律失常的动态特征分析 | 心律类型多，动态模式差异微妙，且阵发性心律失常发作短暂难以捕捉 |
 | 冠脉慢血流无创检测 | 静息心电图无形态学改变，是国际公认的"静默性"心肌缺血病因 |
 | 急诊胸痛风险分层 | 正常ECG的胸痛患者量大、异质性高，最危险的患者常表现最隐匿 |
 
 ### 5.2 研究价值
 
 从临床角度看，我所做的工作在以下两个层面具有明确的价值：
 
 在心血管领域：常规心电图虽是常用的一线检查工具，但其敏感性不足导致大量冠心病患者被漏诊。冠脉造影等金标准方法有创、昂贵、且有辐射风险，不适合大规模筛查和早期检测。我利用CDG提供了一种无创、低成本、操作便捷的补充诊断手段，在多项临床验证中证明了其弥补常规心电图不足的价值。特别是在冠脉慢血流/微血管功能障碍检测这一前沿方向上，我们的工作在国际上率先实现了基于常规心电图的精准无创评估。
 
 在肿瘤心脏病学（Cardio-Oncology）领域：随着放疗技术的进步，肿瘤患者生存率不断提高，但放射性心脏损伤作为远期并发症变得越来越突出。这一问题涉及肿瘤学与心脏病学的交叉，在国际上受到日益关注。我们的研究首次将CDG应用于RIHD的早期检测，并在动物模型中严格验证了CDG早于组织病理学变化的检测能力。考虑到目前临床上缺乏有效的RIHD早期筛查手段，这项工作为癌症患者放疗后的心脏监测开辟了新的可能性，具有显著的临床转化价值和社会意义。
 
 从科学角度看，我将非线性动力学理论和系统辨识方法引入心电信号分析，在传统心电图"形态学"和"时频域"分析范式之外，开创了"动力学"特征这一新维度。CDG在放射心脏损伤中的成功验证，进一步证明了这一方法学范式在多种病因心脏损伤检测中的通用性和灵活性。

 ## 七、小结
 
 回顾2020年以来的研究工作，我认为以下几个方面的成果能够体现我的研究积累和水平：
 
 1. 构建了较大规模的心肌缺血临床心电数据集，其临床代表性优于国际通用公开数据集，为后续算法研究和临床验证提供了数据基础。
 2. 验证并拓展了CDG方法的临床应用价值，将CDG从最初的提出扩展到心肌缺血检测、PCI疗效评估、心律失常分类、放射心脏损伤检测、冠脉慢血流筛查、急诊胸痛风险分层等多个场景。
 3. 实现了CDG从心脏病学到肿瘤心脏病学的跨域拓展，在放射心脏损伤检测这一前沿交叉方向建立了原创性工作，证明了CDG作为通用型心脏损伤早期检测工具的潜力。
 4. 在可解释性方面形成了明确的学术主张，多次在论文中强调"物理意义明确的可解释性"是核心优势，并提出了StackTree、Shapelets等多种可解释性方法。
 5. 研究问题的难度和前沿性持续提升，从"心电图正常能不能检"进展到"能不能鉴别病因"，再到"能不能无创筛查微血管疾病"和"能不能跨域检测放射心脏损伤"，2026年在生物医学工程顶刊IEEE TBME上发表的AG-STDF工作代表了该方向的国际前沿水平。
 6. 最新CDG-MACE评分工作标志着研究从"算法工具"向"临床量化工具"的转化。
